== Description ==

Instagram Gallery by Arrow plugins Creates Instagram Gallery from your Multiple Instagram Accounts or Multiple Hashtags. Show your Instagram Photos & Videos in lightbox. Instagram Gallery feed plugin helps you to display Instagram photos from any private or non-private (public) Instagram account, in save way you can also show instagrm photo feed from any hastag and disply instagram feed into your posts, pages, home or in widget. With instagram feed plugin you can add multiple instagram feeds into single page, post or widget using shortcodes.

== Installation ==


The easy way :

* Go to the Plugins Menu in WordPress.
* Search for plugin "Instagram gallery"
* Click "Install".
* After Installation click activate to start using the Instagram.
* Go to "Instagram Gallery" from Dashboard menu.

Not so easy way :

* To install "Instagram plugin" via FTP
* Download the Instagram plugin
* Unarchive "Instagram plugin"
* Copy folder with ap-instagram-gallery.zip
* Open the ftp \wp-content\plugins\
* Paste the plug-ins folder in the folder
* Go to admin panel => open item "Plugins" => activate Instagram.
* Go to Instagram from Dashboard menu.
